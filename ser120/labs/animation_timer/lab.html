SER120 Labs

Lab 10: Animation Timer

Objectives

  • Implement a button listener
  • Use a timer to animate a swing component

Task 1: Display Button and Text

  1. Create an App class that extends JFrame and displays an empty JPanel.
  2. Add a JLabel with the text "Go Bobcats!" to your JPanel.
  3. Add a JButton with the text "Start" to your JPanel.

Task 2: Animate Text

  1. Now you need to animate the JLabel. For this you will need a timer to control the animation and a listener for the button.
  2. Clicking on the button when its text reads "Start" should...
    1. Change the button’s text to "Stop".
    2. Cause the label’s text to begin changing color. Every half second, the color of the JLabel should change to a randomly generated color.
    3. To set the text color on a JLabel, use the setForeground method.
  3. Clicking on the button when its text reads "Stop" should...
    1. Change the button’s text to "Start".
    2. Cause the label to stop chaning colors. It should remain on the current color rather than resetting.
  4. Note: this task requires you to write TWO different action listener classes! You should have one listener class for the timer, and a different listener class for the button. Think about what has to happen when you click the button (that’s the button listener’s job), then compare that to what has to happen when the timer ticks down (that’s the timer listener’s job).

Task 3: Multiple Animations

  1. Add a second timer that changes the number of exclamation points in the "Go Bobcats!" message. It should cycle through "Go Bobcats!", "Go Bobcats!!", and "Go Bobcats!!!" before returning to "Go Bobcats!" and repeating this cycle.
  2. You can use any interval you want here, but it must be different from the interval used by your color timer. For example, if your color timer triggers every 0.5 seconds, then this could trigger every 0.75 seconds or every 1 second.
  3. You should write a third action listener class for this task. The action you are performing (changing the label’s text) is completely different from the actions your other listeners perform, so this should be a new action listener class. Don’t try to re-use the timer or button listener from task 2.

Review & Submission

I will review all lab assignments after the deadline. You should usually receive feedback within a week of the deadline.
